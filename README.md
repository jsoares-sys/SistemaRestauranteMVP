# SistemaRestauranteMVP

🎯 Qual problema o sistema resolve?

Organização nos pedidos, conseguir realizar o pedido via QRcode.

👤 Quem usa?

Admin, clientes e pessoal da cozinha

🧩 Qual é o MVP (versão mínima funcional)?

Realizar pedido, recebimento dele na cozinha, fechamento simples daquele pedido.

🚫 O que NÃO vai ter agora?

Estoque, integração fiscal, app nativo.
